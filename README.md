# mkitem-guide

本仓库用于归档和发布多个独立的 HTML 页面。  
这些页面各自具有独立的功能或信息展示用途，不针对特定域名或网站。

## 项目简介

`mkitem-guide` 是一个轻量级的页面归档仓库，旨在集中存放与维护多个 HTML 文件。  
每个文件可独立查看，内容涵盖不同主题或工具页面，适合作为静态资源集合使用。

## 目录结构

```
mkitem-guide/
├── index.html          # 主入口页面
├── page1.html          # 示例页面一
├── page2.html          # 示例页面二
└── assets/             # 公共资源文件夹（可选）
    ├── style.css
    └── script.js
```

- 根目录下的 HTML 文件即为已归档的独立页面。
- `assets/` 用于存放页面共享的样式、脚本或图片等资源文件，可根据需要自行组织。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接在浏览器中打开运行。
- 页面之间没有关联依赖，可单独维护或更新。
- 归档内容不涉及具体域名、商业推广或诱导点击信息。

## 维护说明

- 如需新增页面，直接在仓库根目录添加 `.html` 文件即可。
- 建议为每个页面添加清晰的注释或标题，便于识别。
- 更新页面后，请同步更新 `index.html` 中的导航链接（如有）。
- 如使用公共资源，请确保路径正确，推荐使用相对路径。
